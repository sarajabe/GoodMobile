ANDROID

Android Smartphone LTE Instructions:Android version 4.0 (Ice Cream Sandwich) and up

The following instructions are for most Android devices. While your phone’s menu may look different, the information which goes into each of the fields is the same, no matter what model device you have. No data connection is necessary when entering these settings but a data-capable device is required to surf the web and watch videos online.

    • Press “Menu” button on your device
    • Tap “Settings” > “Wireless Controls” or “Wireless and Network” > “Mobile Networks”
    • Make sure “Data Roaming” and “Use Only 2G Networks” are both UNCHECKED
    • Tap Access Point Names
    • Press Menu button and select New APN
    • Enter the following information
    • Name: GoodMobile
    • MMS Proxy: Not set
    • APN: wholesale
    • MMS Port: 8080
    • Proxy: Not set
    • MCC: 310MNC: 260
    • Port: Not set
    • Authentication Type: None
    • Username: Not set
    • APN Type: default,mms,supl
    • Password: Not set
    • APN protocol: Not set
    • Server: Not set
    • Bearer: Unspecified
    • MMSC: http://wholesale.mmsmvno.com/mms/wapenc
    • Press the “Menu” button on your device and then “Save” on the screen.
    • Go to Account Summary to manage your account
